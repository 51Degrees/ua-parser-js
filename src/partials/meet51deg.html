UAParser- 51D.js

A lightweight JavaScript library wrapping 51Degrees device detection cloud service. The lib provides comprehensive device data based on User-Agent Client Hints and/or legacy User-Agent string. Can be used either in a browser (client-side) or node.js (server-side).

Client-side supports detection of the current client or detection from an arbitrary header map, server-side supports detection from a header map only.

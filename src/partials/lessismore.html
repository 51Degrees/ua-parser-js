Less is More

Extensive device data

UA-parser-51d-js provides access to the 51Degrees Cloud service, which is capable of identifying over 200 different device properties, such as platform, browser, engine, CPU, vendors, and many more. The device data is maintained and updated by a professional data analyst team on a regular basis.

Small library footprint

The ua-parser-51d-js library footprint is a tiny 13KB, which is smaller than many other available User-Agent parsers available.

Open source with a freemium model

We recognize the value of open source – that’s why the ua-parser-51d-js library is freely available with a limited set of data properties. If you require extra data points or more requests per month, we offer different pricing plans to suit your needs.

Client-side & Server-side

Whether it's a client-side webpage running in a browser or a server-side node.js app, simply type:

$ npm i ua-parser-51d-js --save
